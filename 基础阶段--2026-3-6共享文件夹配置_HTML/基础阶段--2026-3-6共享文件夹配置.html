2.在windows主机D盘下新建一个文件夹share,再点击添加

如下配置：

添加按钮按下后，出现安装向导，点击【浏览】：选择的是主机真正物理磁盘的某个文件夹
3.下一步之后开启虚拟机
cd /mnt进入挂载目录
su root输入管理员密码
在mnt文件夹下新建hgfs文件夹
mkdir -p /mnt/hgfs
虚拟机的共享文件夹在 /mnt/hgfs
首次配置需要安装vmhgfs-fuse工具
命令是：apt install open-vm-tools
（或者# 安装 VMware 共享文件夹核心工具（包含 vmhgfs-fuse）sudo apt install -y open-vm-tools open-vm-tools-desktop）
4.输入vmhgfs-fuse .host:/share /mnt/hgfs -o allow_other


注意：如果执行该命令的路径在/mnt/hgfs下，直接ls查看共享文件夹是否已经成功，有可能刷新不出来。
解决方法：cd .. 回到上一级目录/mnt，再cd hgfs重新进入到hgfs目录下，ls可以看到共享文件夹

挂载点hgfs不存在，主动在/mnt下执行mkdir hgfs创建，再挂载

1.ssh配置
参考博客：https://developer.aliyun.com/article/1570236
第一步：
在终端中，输入以下命令以安装SSH服务器软件包：
sudo apt update
sudo apt install openssh-server

第二步：
安装完成后，SSH服务器将自动启动。您可以使用以下命令检查SSH服务器运行状态：
sudo systemctl status ssh

第三步：
如果您的系统启用了防火墙（例如ufw），您可能需要打开SSH端口（默认为22）以允许远程连接。您可以使用以下命令打开SSH端口：
sudo ufw allow ssh

第四步：
现在，您可以使用SSH客户端连接到您的Ubuntu服务器。在另一台计算机上，打开终端并输入以下命令：
ssh username@server_ip_address

2.windows远程登录
我们使用RDP和xfce方案：
第一步：Ubuntu 端配置

# 1. 安装 Xfce 轻量桌面（对 xrdp 100% 兼容）
sudo apt install xfce4 xfce4-goodies -y
# （安装时弹出「默认显示管理器」选择，保持 gdm3 不变）

# 2. 配置 xrdp 启动 Xfce 桌面
echo "startxfce4" > ~/.xsession
chmod +x ~/.xsession

# 3. 重启 xrdp 服务
sudo systemctl restart xrdp

# 4. 关键：在 VMware 窗口里注销本地用户 luochen
# 操作：VMware → 右上角用户名 → 注销（回到登录界面）

第二步：
Windows 端连接（用自带「远程桌面连接」）
按 Win + R，输入 mstsc，打开「远程桌面连接」；
输入 Ubuntu IP：192.168.46.132，点击「连接」；
在弹出的 xrdp 登录窗口中：
Module：选择 Xorg
Username：输入 luochen
Password：输入 Ubuntu 的登录密码

成了！！！
