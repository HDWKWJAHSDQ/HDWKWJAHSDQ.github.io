思路：

for循环
找到最大元素的索引
这个元素开始 全部往后挪b的长度位置
最后把字符串b放进去

abecd
abe a cd
代码：
#include <stdio.h>

int main()
{
char a[10000];
int i, j;
char b[5] = "abc";
gets(a);
int index = 0;
for(i = 0; a[i] != '\0'; i++)
{
if(a[i] > a[index])
index = i;
}
for(i = index + 1; a[i] != '\0'; i++)
a[i - 3] = a[i];
a[i - 3] = '\0';
for(i = 0; i < 3; i++)
a[index + 1 + i] = b[i];
puts(a);
}
