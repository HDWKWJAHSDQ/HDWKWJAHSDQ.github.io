练习题：

条件运算符：（三目运算符）
?: 带参数的宏
int max = (a>b) ? a : b; // a>b为真，max=a；否则max=b
一、条件运算符（三目运算符 ?:）
核心：简化版 if-else，一行搞定简单判断语法：条件 ? 满足执行 : 不满足执行示例：c运行int max = (a>b) ? a : b; // a>b为真，max=a；否则max=b

二、带参数的宏（结合三目）
核心：预编译文本替换，无运行开销，封装简单逻辑语法：#define 宏名(参数) 替换内容示例（找最大值）：c运行
#define MAX(a,b) ((a)>(b)?(a):(b)) // 必须加括号避坑
// 调用：MAX(10,20) → 直接替换成 ((10)>(20)?(10):(20))

总结
1. 三目运算符：简单条件判断，比 if-else 简洁；
2. 带参数宏 + 三目：封装简单逻辑（如最值），编译前替换，记得加括号防出错。

#include <stdio.h>

{
int a,b;

printf("%d\n",res/100);
else
printf("%d\n",a+b);
return 0;
