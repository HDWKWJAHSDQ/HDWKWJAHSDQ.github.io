break;
swap(i,j,arr);
l++;
r--;
}
}

、核心问题：{} 不是函数，只是限制作用域（栈空间解释）

1. 什么是「块作用域」？
C 语言中，用 {} 包裹的代码叫代码块（复合语句），它不是函数，核心作用是限制变量的作用域：• 块内定义的变量（如 int b=4,c;），只在这个 {} 内有效
• 块执行结束，变量立即销毁，不会影响块外的同名变量
• 块内可以访问块外未被屏蔽的变量（如 a，块内没重新定义 a，所以用外层 a）

2. 栈空间的分配逻辑

（1）块内变量的存储：栈上分配，和函数局部变量完全一致
• 当程序执行到 {} 块时，块内定义的 b=4、c 会在栈上分配空间
• 块执行完（} 结束），栈上的局部变量空间自动释放（销毁），和函数调用结束销毁局部变量的逻辑完全一样
• 区别：函数是独立的调用栈帧，块是在当前函数的栈帧内，额外开辟临时空间
